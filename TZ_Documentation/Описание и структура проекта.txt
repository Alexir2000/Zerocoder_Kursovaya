Проект "flowers_shop" представляет собой веб-приложение для заказа цветов с интеграцией через Telegram бота. Пользователи могут регистрироваться, просматривать каталог товаров, добавлять товары в корзину, оформлять заказы, оставлять отзывы и получать аналитику по заказам. Администраторы могут управлять заказами и просматривать аналитические данные.
Структура проекта

Проект состоит из нескольких приложений (фактическая структура):

    main - Основное приложение, включающее модели пользователей и другие основные данные.
    catalog - Приложение для управления каталогом товаров.
    orders - Приложение для управления заказами.
    reviews - Приложение для управления отзывами и рейтингами.
    analytics - Приложение для управления аналитикой и отчетами.
    scripts - Сценарии для автоматизации и управления данными.
