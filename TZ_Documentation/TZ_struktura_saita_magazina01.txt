Общее описание проекта

Проект "flowers_shop" представляет собой веб-приложение для заказа цветов
с интеграцией через Telegram бота. Пользователи могут регистрироваться,
просматривать каталог товаров, добавлять товары в корзину, оформлять заказы, оставлять отзывы и получать аналитику по заказам. Администраторы могут управлять заказами и просматривать аналитические данные.
Структура проекта

Проект состоит из нескольких приложений:

    main - Основное приложение, включающее модели пользователей и другие основные данные.
    catalog - Приложение для управления каталогом товаров.
    orders - Приложение для управления заказами.
    reviews - Приложение для управления отзывами и рейтингами.
    analytics - Приложение для управления аналитикой и отчетами.

Шаблоны и HTML файлы

main
    templates/main/
        index.html - Главная страница приложения.
        login.html - Страница входа пользователя.
        register.html - Страница регистрации пользователя.
        password_reset.html - Страница сброса пароля.
        password_reset_done.html - Страница уведомления о сбросе пароля.
        password_reset_confirm.html - Страница подтверждения сброса пароля.
        password_reset_complete.html - Страница уведомления об успешном сбросе пароля.

catalog
    templates/catalog/
        catalog.html - Страница для просмотра каталога товаров.

orders
    templates/orders/
        cart_detail.html - Страница для просмотра содержимого корзины.
        checkout.html - Страница оформления заказа.
        admin_order_list.html - Страница для администрирования списка заказов.
        change_order_status.html - Страница для изменения статуса заказа.

reviews
    templates/reviews/
        add_review.html - Страница для добавления отзыва.
        tovar_reviews.html - Страница для просмотра отзывов о товаре.

analytics
    templates/analytics/
        analytics.html - Страница для просмотра аналитических данных и отчетов.

Модели данных

main
    StatusDostupa - Модель для хранения статусов доступа пользователей.
    Users - Модель пользователей.
    Tip_Tovara - Модель типов товаров.
    Kat_Tovara - Модель категорий товаров.
    Tovar - Модель товаров.
    StatusZakaza - Модель статусов заказов.
    Zakaz - Модель заказов.
    BaseOtziv - Модель отзывов.
    Otchet - Модель отчетов.

orders
    CartItem - Модель элементов корзины.
